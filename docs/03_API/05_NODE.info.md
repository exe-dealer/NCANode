# NODE.info

Возвращает информацию о сервере

Пример запроса:

```json
{
	"version": "1.0",
	"method":"NODE.info"
}
```


Примерный ответ:
```json
{
    "result": {
        "dateTime": "2018-10-23 00:09:37",
        "timezone": "Asia/Almaty",
        "name": "NCANode v1.0",
        "version": "1.0"
    },
    "message": "",
    "status": 0
}
```
